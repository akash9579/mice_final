Mice Protien Prediction

This project in based on multi-classification of Mice protein expression data . We build end to end project till deployment on Google cloud platform. and on client side as well. For api creation we used Flask and jinja. For classification we used three different techniques random forest,decision tree,KNN. we also used dask dataframe instead of pandas dataframe in the final deployment module(dask dataframe support parallel processing).we also implement logging system for maintaining log records of the application.
All this project code, development versions,you will find in ineuron repository.
